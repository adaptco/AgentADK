# Matrix Dash

A standalone HTML5 Canvas web game with Geometry Dash style one-button runner mechanics and Matrix-inspired visuals.

## Run

From the repository root:

```powershell
python -m http.server 8000
```

Then open:

`http://localhost:8000/matrix_dash/`

## Controls

- `Space`, `Arrow Up`, `W`
- Click/tap anywhere on the canvas
- Mobile jump button in the bottom-right
